# ProfileSettings - Missing Functionalities

## Current New Version Has:
- ✅ Modern tabbed interface (Account, Profile, Security)
- ✅ Basic fields (name, email, phone, location)
- ✅ Social links (LinkedIn, GitHub, Website)
- ✅ Password change
- ✅ Account deletion
- ✅ Photo upload with better error handling
- ✅ Admin badge display

## Missing from Old Version:

### 1. Role-Specific Sections
- ❌ **Domestic Student Profile** (university, major, year, clubs, peer mentorship, campus buddy, financial aid)
- ❌ **International Student Profile** (home country, visa status, arrival date, housing needs, cultural adjustment)
- ❌ **Alumni Profile** (graduation year, current employer, industry, years experience, job posting ability)
- ❌ **Professional Profile** (company details, industry, size, bio, internships, webinars)

### 2. General Preferences Section
- ❌ Campus involvement level
- ❌ Languages spoken
- ❌ Notification preferences (campus events, mentorship requests, community updates)

### 3. Mentor System
- ❌ "Become a Mentor" application button
- ❌ Application status display (pending, approved, rejected)
- ❌ "You're a Verified Mentor" message
- ❌ Mentor toggle switch
- ❌ Mentor expertise field
- ❌ Mentor bio field
- ❌ Link to mentorship dashboard

### 4. Complete Profile Section
- ❌ Bio textarea
- ❌ Skills input
- ❌ Interests input
- ❌ Work experience textarea
- ❌ Education textarea

### 5. Settings & Features
- ❌ Resume Auto-Save toggle

### 6. Student-Specific Features
- ❌ Mentorship preferences (offer peer mentorship, campus buddy, willing to mentor international students, event hosting)

### 9. International Student Features

## Action Required:
The new ProfileSettings needs to be completely rewritten to include ALL old functionalities while maintaining the modern design aesthetic.

## Recommendation:
Keep the new modern design but restore ALL sections and fields from the old version. The page should have:
1. Account Overview (with photo, basic info, role-specific fields)
2. General Preferences (campus involvement, languages, notifications)
3. Role-Specific Profile Section (based on user role)
4. Mentor System (application, status, toggle, profile)
5. Complete Profile (bio, skills, experience, education, etc.)
6. Resume Auto-Save toggle
7. Role Privileges display
8. All modals (password, email, delete)
